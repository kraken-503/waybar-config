## Waybar
[Waybar](https://github.com/Alexays/Waybar) is a highly customizable status bar for Linux, designed to work seamlessly with Wayland compositors like Sway and Hyprland. It provides modules for system information, application status, and custom scripts, allowing you to tailor the bar to your workflow. With support for JSON configuration and CSS styling, Waybar makes it easy to create a sleek, personalized interface that fits perfectly into your desktop environment.
